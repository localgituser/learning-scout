# Contributing

Contributions are welcome. Please open an issue before submitting a PR for significant changes.

By submitting a pull request, you certify that:

1. You have the right to submit the contribution under this project's license ([PolyForm Noncommercial 1.0.0](LICENSE)).
2. You grant the project author a perpetual, irrevocable license to use your contribution under the same or any future license this project adopts.

This project follows the [Developer Certificate of Origin v1.1](https://developercertificate.org/).

## Guidelines

- Run `pytest tests/ -q` before submitting — all tests must pass.
- Keep changes focused; one concern per PR.
- No new dependencies without discussion.
